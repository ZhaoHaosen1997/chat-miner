============================================
  Chat-Miner 新手教程
  群聊内容分析工具 v1.0
============================================

Chat-Miner 是一个群聊分析工具。导入微信/QQ 聊天记录后，
AI 会自动生成每日报告、群友画像、周报月报年报等有趣内容。

============================================
  第一步：导出聊天记录
============================================

【微信】使用 WeFlow 工具导出：
  1. 下载 WeFlow: https://github.com/Shuax/WeFlow
  2. 打开 WeFlow，选择要导出的群聊
  3. 导出格式选择 "ArkMe JSON"
  4. 导出后会得到一个 .json 文件

【QQ】使用 qq-chat-exporter 导出：
  1. 下载: https://github.com/shuakami/qq-chat-exporter
  2. 选择 V5+ 版本，导出 JSON 格式
  3. 私聊和群聊都可以导入

============================================
  第二步：导入数据
============================================

  1. 双击 ChatMiner.exe，会弹出一个黑框（服务器窗口）
  2. 浏览器会自动打开，进入 ChatMiner 首页
  3. 在首页点击 "上传聊天记录" 按钮
  4. 选择你导出的 .json 文件
  5. 使用完毕后，关闭黑框即可退出程序

============================================
  第三步：配置大模型（最重要！）
============================================

Chat-Miner 需要大模型来生成分析报告。推荐使用在线 API，
只需要一个 API Key 就能用，不需要自己部署。

【推荐方案：DeepSeek 在线 API】
  1. 打开 https://platform.deepseek.com
  2. 注册账号，在 "API Keys" 页面创建一个 Key
  3. DeepSeek 价格很便宜，充 10 块钱能用很久
  4. 在 Chat-Miner 左侧菜单点击 "设置"
  5. 找到 "在线模型"，粘贴你的 API Key
  6. 点击模型名称旁边的 "测试连接" 按钮
  7. 显示 "连通成功" 就完成了！

【也可以使用其他在线 API】
  在设置页点击 "新增"，填写：
  - 名称：随便起
  - 类型：在线
  - 端点：API 地址（如 https://api.openai.com/v1/chat/completions）
  - API Key：你的 Key
  - 模型名：如 gpt-4o

【高级用户：本地 Ollama】
  如果你已经安装了 Ollama，在设置页的 "Ollama (本地)"
  中就可以直接使用。但本地模型需要较好的显卡。

============================================
  第四步：开始分析
============================================

配置好模型后，回到首页：
  - 点击日历上的日期 → "开始 AI 分析" → 等待生成日报
  - 或者点击 "一键分析全部" 批量生成所有日期的日报
  - 左侧菜单可以查看 "群友画像"、"周报"、"月报" 等

============================================
  常见问题
============================================

Q: 显示 "API Key 无效" 或 "余额不足"？
A: 检查 Key 是否正确粘贴，确认账户有余额。

Q: 生成报告很慢？
A: 在线模型通常 10-30 秒完成。如果超时，请检查网络。

Q: 能分析多个群吗？
A: 可以。在首页左上角可以切换群，也可以上传多个群的记录。

Q: 聊天记录安全吗？
A: 数据全部存储在你的电脑上（ChatMiner 同级 data 目录）。
   使用在线 API 时，聊天文本会发送到 API 服务器进行处理。

Q: 如何升级到新版本？
A: 便携版：解压新版 zip 覆盖旧目录即可，数据不会丢失。
   安装版：运行新版 setup.exe，数据不会丢失。

Q: 如何退出程序？
A: 关闭弹出的黑框窗口即可。下次使用再次双击 ChatMiner.exe。

Q: 从旧版（源码版）迁移数据？
A: 将旧版的 data 文件夹复制到 ChatMiner 文件夹中即可。
   启动时会自动备份和兼容旧数据库。
   如果遇到报错，删除 data\chat_miner.db 重新导入。
