## ADDED Requirements

### Requirement: 全服统一天气
系统 SHALL 基于日期生成全服统一的天气，所有群共享同一天气。

#### Scenario: 天气种子不含群ID
- **WHEN** 系统生成每日天气
- **THEN** 天气种子仅依赖日期字符串（`f"weather_{date_str}"`），不包含 group_id

#### Scenario: 所有群天气一致
- **WHEN** 同一天不同群触发每日结算
- **THEN** 所有群获得相同的天气类型和效果

### Requirement: 7种天气类型
系统 SHALL 从 7 种天气中按概率随机生成每日天气。

#### Scenario: 天气概率分布
- **WHEN** 系统生成天气
- **THEN** 按以下概率分布：☀️晴天 40%、🌧️小雨 20%、⛈️暴风雨 10%、🌈彩虹 5%、🌈🌈双彩虹 3%、🌪️沙尘暴 7%、🌠流星雨 2%、其余概率为 ☀️晴天兜底

#### Scenario: 双彩虹效果
- **WHEN** 天气为"双彩虹"
- **THEN** 全体鱼 happiness +20

#### Scenario: 沙尘暴效果
- **WHEN** 天气为"沙尘暴"
- **THEN** 全体鱼当日检定获得 -1 修正，但宝藏事件概率 +10%

#### Scenario: 流星雨效果
- **WHEN** 天气为"流星雨"
- **THEN** 流星许愿事件必定触发且 DC -5

### Requirement: 天气每日更换
系统 SHALL 每天仅更换一次天气（跟随每日结算），不更频繁变更。

#### Scenario: 同一天多次触发不换天气
- **WHEN** 同一天内被动事件多次触发
- **THEN** 天气保持不变（每日结算仅在当天首次触发时执行）

### Requirement: 天气前端动画
前端 SHALL 在 FishTank 视图中渲染当前天气的 CSS 动画叠加层。

#### Scenario: 天气动画叠加
- **WHEN** 鱼塘页面加载并获取到当前天气
- **THEN** FishTank 上方渲染天气对应的半透明 CSS 动画层（雨：垂直雨滴 / 暴风雨：斜雨+闪电闪烁 / 彩虹：顶部弧形光晕 / 双彩虹：双层光晕 / 沙尘暴：水平沙粒 / 流星雨：间歇对角线光点）
