## v1.19 — 底座重构 🚧

> 基于 v1.18.8 全量审计，统一消息格式化、分词、采样策略

### P0 — 中文分词修复
- 删除 4-gram 正则切片 `re.findall(r'[一-鿿]{2,4}', text)`
- 引入 jieba 精确模式 + 自定义词典（梗百科词条、群名片）
- 作为离线兜底；AI 管线关键词由 AI 自己产出

### P1 — 消息格式化共享层
- 新建 `services/message_formatter.py`：统一 PII 过滤 + 内容过滤 + stable_id + 截断
- 各管线改为调用共享函数，消除 7 套独立实现
- 顺修：移除视频消息、引用消息源头排除、画像补 PII、梗百科统一 stable_id

### P2 — 采样策略重做
- 删除 `_score_message_interestingness`（正则选"热闹"消息无意义）
- 梗百科：去人去重，按时间窗口取最近 N 条
- 周报：7 天全量消息
- 月报：删 `sampled_chat`，改用每日日报 one_line 摘要
- 年报：月报 headline 摘要
- 金字塔压缩模型：日报 → 周报 → 月报 → 年报

### P3 — 可配置 Prompt 补全
- 周报/月报/年报补 `get_default_prompt()` 调用（当前 UI 能配置但不生效）
- 梗百科加可配置 Prompt 支持

### P4 — 过滤词生效范围修正
- 从仅词频统计 → 消息内容层面也过滤后发给 AI
- UI 描述更新

### P5 — AI 产出关键词
- 周报/月报/年报/画像已有 AI 调用，输出 JSON 加 `keywords` 字段
- 零额外 API 调用，取代 Python n-gram 碎片
