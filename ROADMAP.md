# Chat-Miner 版本路线图

## v0.6 — 群聊考古 + 可视化

> 把已有的数据用更好玩的方式呈现

### 群聊热搜榜 🔥
- 微博热搜格式：每日/每周话题热度排行
- 自动生成话题标签（`#张三又发红包了` `#今天的午餐之争`）
- 带热度指数、上榜天数、涨跌标记（`↑3` `↓2` `新` `沸`）

### 关系可视化 🕸️
- 群友互动节点图（谁和谁聊最多、谁@谁最多）
- 线条粗细=互动次数，颜色=关系类型
- 支持按时间范围筛选（本周/本月/全部）

### 群友"黑历史"考古 🏺
- 翻出某个群友的第一条发言、最搞笑发言、最长发言
- "X 年前的今天"回忆杀
- AI 给每个群友写考古报告："据史料记载，张三于2023年3月首次出现在本群..."

## v0.7 — 周报/月报 + 在线模型 ✅

> 引入 DeepSeek API 做深度推理，本地 14B 负责脱敏消化

### 周报/月报 📊
- ✅ 自然周/自然月报告（ISO 8601 周、自然月）
- ✅ 历史回溯：任意有足够日报数据的周/月都可生成
- ✅ 以**日报摘要**为输入（不是原始聊天）
- ✅ 内容：话题全景、情绪过山车、名场面回顾、周/月展望
- ✅ 月报更深：氛围诊断、群友聚光灯、上月对比

### 在线模型配置 ☁️
- ✅ DeepSeek API 集成（DeepSeek Chat + DeepSeek Reasoner）
- ✅ `.env` 配置 key 和模型名
- ✅ 降级策略：DeepSeek 不可用时自动降级到本地 14B

### 脱敏架构 🔒
```
原始聊天 ──→ 本地 14B ──→ 日报摘要 ──→ DeepSeek ──→ 周报/月报
(私密)      (消化)     (脱敏)        (深度推理)    (产出)

DeepSeek 看到的：                     DeepSeek 看不到的：
  6月1日：话题3个，情绪😄               任何原始聊天内容
  关键词：火锅, 开黑, 周五               wxid、头像URL
  活跃15人，名场面点评                   具体发言原文
```
本地 14B 先消化原始聊天 → 产出结构化摘要 → DeepSeek 只看摘要，看不到原始对话。

## v0.8 — 年度盛典 + 智能问答

### 年度颁奖典礼 🏆
- 自动生成 10+ 奖项和提名名单，AI 写颁奖词
- 奖项示例：
  - "年度金句王" — 发言被引用次数最多
  - "深夜守夜人" — 凌晨发言占比最高
  - "表情包终身成就奖" — emoji 种类最多
  - "最佳新人奖" — 今年入群且活跃度飙升
  - "潜水冠军" — 在线但几乎不发言
  - "年度 CP 奖" — 互动最频繁的一对

### 智能问答 💬
- "这个群最常讨论什么话题？""最近一个月谁最活跃？"
- "张三和李四什么时候开始互怼的？"
- 基于日报摘要做 RAG（不碰原始聊天）
