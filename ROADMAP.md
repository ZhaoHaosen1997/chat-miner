## v0.8 — QQ 聊天记录导入 ✅

> 支持导入 QQChatExporter V5 导出的 JSON 格式

### 核心设计
- 在 `ParsedChat.load()` 入口做格式检测 + QQ 适配
- 将 QQ JSON 归一化为与微信完全相同的内部结构
- 下游分析管线（日报/周报/月报/画像）零改动复用

### 改动范围
- `services/parser.py`：新增 `_detect_qq_format()` + `_normalize_qq()`，修改 `load()`
- `frontend/src/components/UploadModal.vue`：文字改为"微信/QQ聊天记录"

### 技术要点
- QQ 格式检测：顶层有 `metadata` + `chatInfo`（微信没有）
- 消息类型映射：QQ type_1→文本, type_3→引用, type_7→系统 等
- 发送者归一化：从消息中提取 UID 分配 senderID
- 私聊处理：双方 UID 拼接作为群标识

详见 `plan-hazy-stroustrup.md`

## v1.1 — 群鱼塘 🐟

> 把群聊活跃度变成养鱼游戏，聊天即养鱼

### 核心玩法

**领养**
- 群友发送 `/领养` 或首次发言时自动获得一条鱼苗
- 鱼有品种（随机）：金鱼🐟、锦鲤🎏、小丑鱼🤡、斗鱼🐠、龙鱼🐉 等
- 鱼有稀有度：普通(白)、稀有(蓝)、史诗(紫)、传说(金)
- 可以给鱼起名字，不命名默认叫"XX的鱼"

**养成**
- 每日发言自动喂食：每天发言 ≥5 条 = 自动投喂一次
- 特定指令加速成长：
  - `喂饲料` — 手动喂食，+10 成长值（每天限 3 次）
  - `换水` — 清洁鱼缸，+5 幸福值
  - `摸鱼` — 摸一摸，+2 亲密度
- 连续活跃天数 = 暴击加成（7 天连签双倍成长）
- 长期潜水（3 天不说话）→ 鱼变瘦，幸福值下降

**进化**
- 成长值达到里程碑自动进化：鱼苗 → 幼鱼 → 成鱼 → 大鱼 → 传说
- 每个阶段外观变化（前端 CSS 动画）
- 传说级别解锁特殊称号和颜色

**互动**
- `晒鱼` — 在群里展示自己的鱼（生成卡片消息）
- `喂XX的鱼` — 帮别人喂鱼（+社交值）
- `斗鱼 @XX` — 和指定群友比鱼的大小，胜者得荣誉
- `鱼塘` — 查看全群鱼塘总览（排行榜）

**排行榜**
- 最大鱼王 🏆
- 最幸福鱼 😊
- 最佳饲养员（喂别人鱼最多）
- 稀有度排行

### 数据存储
- 新表 `fish_pond`：group_id, wxid, fish_name, fish_type, rarity, growth, happiness, intimacy, stage, created_at
- 每天凌晨结算：自动喂食 + 潜水惩罚
- 消息 Hook：解析特定指令触发操作

### 趣味机制
- 🎂 生日鱼：鱼被领养满 30 天触发"成年礼"
- 🌧️ 天气事件：随机"雨天"额外成长、"暴风雨"需要手动换水
- 🎣 周末钓鱼大赛：周六周日发言双倍饲料
- 🦈 鲨鱼来袭：极低概率事件，随机吃掉一条幸福值最低的鱼（被吃可重新领养，获得稀有鱼概率翻倍）

### 前端
- 新页面 `/fishpond`：全群鱼塘俯瞰（鱼游动动画）
- 个人鱼卡片：可分享到群聊的 SVG/Canvas 图片
- 排行榜实时更新
